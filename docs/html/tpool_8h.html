void tpool_finish ( tpool_t *  pool)

Finish the thread pool execution.

Once this is executed, the thread pool stops accepting tasks, completes all of the waiting tasks, and then cleans up all of the memory that it assigned internally for executions including destroying all of the threads in the thread pool.

Parameters
poolThread pool instance.
